# TECH TITANS - Grocery Data Pipeline

##  Overview
This project implements an end-to-end data pipeline using Snowflake.

##  Architecture
RAW → VALIDATION → SILVER → DWH

##  Features
- Data Ingestion (CSV → Snowflake)
- Data Validation Layer
- Dim & Fact Tables
- SCD Type 2 Implementation
- Anomaly Detection
- KPI Metrics
- Streams & Tasks (Automation)

##  KPIs
- Stockout Rate
- Inventory Turnover
- Category Sales
- Revenue per Store
- Average Discount
